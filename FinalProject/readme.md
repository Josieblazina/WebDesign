Something new I learned while building this site was how to work with Bootstrap components and integrate them with my own custom CSS without the two fighting each other. Getting the navbar to look like mine instead of a default Bootstrap navbar took some figuring out, but once I understood how to override the styles it started to feel natural.
Balancing creativity with usability mostly came down to making sure the things that looked interesting also made sense to navigate. The tile grid on the homepage is a good example of that. It is visually the part of the site I am most proud of, but it also functions as the main navigation, so every design choice I made to it had to serve both goals at once.
What I am most proud of is how cohesive the site ended up feeling overall. The color palette, typography, image choices, and layout all feel like they belong to the same thing, and that is something I was conscious of the whole time I was building it. It does not feel like a class assignment to me, it feels like something I would actually use and show people.
As for using these skills later, I am genuinely planning to keep building this out as a real portfolio site. Understanding how to structure and style a multi-page site from scratch, manage files, and deploy through GitHub are things I can see myself using whether I am presenting work to someone, applying for something, or just continuing to develop projects on my own.
